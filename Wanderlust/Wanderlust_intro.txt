APIXU API: 
- Apixu provides access to weather and geo data via a JSON/XML restful API. 
  It allows developers to create desktop, web and mobile applications using this data very easy.

    We provide following data through our API:
    Real-time weather
    10 day weather forecast
    Astronomy
    Time zone
    Location data
    Search or Autocomplete API
    NEW: Historical weather
